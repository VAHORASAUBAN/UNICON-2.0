CSV Format

Please ensure your CSV file has the following headers:

Field Description
enrollment Unique enrollment number of the student
firstname First name of the student
middlename Middle name of the student (optional)
lastname Last name of the student
email Email address of the student
password Password for student account
mobile_number Mobile number of the student
gender Gender of the student
birth_date Birthdate of the student
address_line_1 First line of the student's address
address_line_2 Second line of the student's address (optional)
country Country of the student
state State of the student
city City of the student
pincode Pincode of the student's location
student_department Department the student belongs to (updated)
course Course the student is enrolled in (updated)
semester Current semester of the student
division Division the student is placed in
student_image Profile image of the student (updated)
